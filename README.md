# rclone-choco

A Chocolatey package for [RClone Manager](https://github.com/RClone-Manager/rclone-manager), a simple cross-platform GUI for [rclone](https://rclone.org/), the command-line program to manage files on cloud storage.

## Description

RClone Manager provides an easy-to-use graphical interface for rclone, allowing you to manage your cloud storage files without dealing with the command line. This Chocolatey package makes it simple to install RClone Manager on Windows systems.

## Installation

Install RClone Manager using Chocolatey:

```powershell
choco install rclone-manager
```

## Requirements

- Windows 7+ (64-bit or ARM64)
- Chocolatey package manager

## Features

- Cross-platform GUI for rclone
- Support for all major cloud storage providers
- Easy configuration and management of remotes
- File browsing and transfer operations

## Usage

After installation, launch RClone Manager from the Start menu or desktop shortcut. For detailed usage instructions, refer to the [RClone Manager Wiki](https://github.com/RClone-Manager/rclone-manager/wiki).
